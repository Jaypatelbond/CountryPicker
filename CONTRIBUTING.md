# Contributing to Country Picker

Thank you for your interest in contributing to the Country Picker library! We welcome all contributions, from bug reports to feature requests and code changes.

## ü§ù How to Contribute

### 1. Reporting Bugs

If you find a bug, please open an issue on GitHub. Include:
- A clear title and description.
- Steps to reproduce the issue.
- Expected vs. actual behavior.
- Screenshots or logs if applicable.
- Your Android version and device.

### 2. Suggesting Features

Have an idea? Open an issue with the "enhancement" label. Describe:
- The feature you'd like to see.
- Why it would be useful.
- Examples of how it should work.

### 3. Submitting Code Changes

1.  **Fork the Repository**: Click the "Fork" button on the top right of the repo page.
2.  **Clone your Fork**:
    ```bash
    git clone https://github.com/YOUR_USERNAME/CountryPicker.git
    ```
3.  **Create a Branch**:
    ```bash
    git checkout -b feature/your-feature-name
    ```
4.  **Make Changes**: Write your code and tests. Ensure everything builds correctly.
5.  **Commit Changes**:
    ```bash
    git commit -m "Add feature: description of your changes"
    ```
6.  **Push to GitHub**:
    ```bash
    git push origin feature/your-feature-name
    ```
7.  **Open a Pull Request**: Go to the original repository and click "New Pull Request".
